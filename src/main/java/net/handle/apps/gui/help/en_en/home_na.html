"Homing" a Prefix

"Homing" a prefix on a particular site tells the server(s) that make up the site that they are responsible for the given prefix. This way, when a resolver comes along and asks for a handle with that prefix, the server can say "here it is" or "it doesn't exist" or even "Why are you asking me? I don't have it."

If you enter the prefix handle as well as the address and port number of one of the primary servers for the desired site, this tool will "home" the given prefix to that site. A message will be sent to each server in the site indicating that that site will now be responsible for the given prefix. From then on that server will accept and handle requests for the given prefix.

To "home" a prefix, select 'Server Admin' from the main Handle Administration Tool menu, then click 'Home Prefix'.
