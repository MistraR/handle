Remove Handle Help

Only authenticated users can remove an existing handle.

To remove a handle:

  1. Input handle name (Prefix/LocalName)

    Type a handle in the 'Handle' text box. Press 'Return'. This will check the authentication information. If you have not authenticated yet, an authentication box will pop up. To change the authentication information, select 'Setup' from the main Handle Admin Tool menu, then click 'Authentication'.

    After authentication approval, all handle values will be displayed in the 'Handle Data View' field. Without proper authentication only the 'public read' values will be displayed.

  2. Submit the handle for removal

    Press the 'Remove' button to delete the handle. This will respond with an indication of success or failure.
