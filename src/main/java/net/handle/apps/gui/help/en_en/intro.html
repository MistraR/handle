Handle.net Administration Tools

Welcome to the Handle.net Administration Tool. This tool was developed for current users of CNRI's Handle.net Software. The functions listed below allow authorized users (Administrators) to create, modify, and delete handles as well as allow general users to resolve handles. The Help text can be accessed under each of the functions by pressing the Help button at the bottom of the active window. The complete set of Help instructions is also available on the Handle web site (www.handle.net).
