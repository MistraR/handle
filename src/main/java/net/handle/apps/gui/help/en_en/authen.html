Authentication

In order to authenticate, the user will need a handle value and its associated secret key or private key.

To authenticate:

  1. Select 'Setup' from the main Handle Administration Tool menu.
  2. Select 'Authentication'.
  3. Select the Authentication Type of 'Secret Key' or 'Public Key'.
  4. Enter the 'ID Handle' and 'ID Index'.
  5. Enter the 'Private Key File' path or the 'Secret Key'.
  6. Click 'OK' to begin authentication or click 'Cancel' to cancel it.
